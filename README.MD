# Microservice Architecture


Components:
* [Express JS for API Server](https://expressjs.com/)
* [MySQL for Relational Data Management](https://www.mysql.com/)
* [Redis for Pub/Sub & Cache Management](https://redis.io/)
* [React Application for Client](https://react.dev/)
* [Worker Redis Consume/Subscribe](https://nodejs.org/en)


#### Please follow this instruction

```bash
git clone git@github.com:ravenKaisar/express-mysql-redis-react.git
```
```bash
cd express-mysql-redis-react
```
```bash
cp .env.example .env
```
```bash
docker-compose build
```
```bash
docker-compose up
```
